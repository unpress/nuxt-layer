## Usage

### Strong

::stack
  :::div{.p-4}
  **Just a strong paragraph.**
  :::

```md
**Just a strong paragraph.**
```
::

### Italic

::stack
  :::div{.p-4}
  _Just an italic paragraph._
  :::

```md
_Just an italic paragraph._
```
:: 


## Tips

You can also use this syntax in the page editor.
